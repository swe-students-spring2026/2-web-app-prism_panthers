## User stories

- As a applicant, I want to log a new job application with the company name, role, and date applied, so I can keep a record of everywhere I’ve reached out to.

- As a job seeker, I want to view a summary list of all my active applications, so I can quickly see my current progress.

- As a candidate preparing for an interview, I want to view the specific notes and job description I saved for a company, so I don't mix up details during the call.

- As a applicant, I want to update the status (e.g., from "Applied" to "Interviewing"), so my dashboard accurately reflects where I stand in the hiring process.

- As a user who made a mistake, I want to edit the details of an existing entry (like fixing a typo in the company name), so my data remains professional and accurate.

- As a rejected or uninterested candidate, I want to delete or archive an application from my list, so I can keep my mobile screen focused only on active opportunities.

- As a applicant with multiple applications, I want to search for a company by name, so I don't have to scroll through a long list to find the one I'm looking for.

- As a user concerned about privacy, I want to log in to my own private account, so that other people cannot see which companies I am applying to.
